# Thrillist 📝  
Live Blog Updating Platform  

---

## 🚀 Overview  
Thrillist is a full-stack **MERN-based blog platform** where users can read, write, and interact with blogs in real-time.  
Designed for smooth performance, clean UI, and engaging user experience.  

---

## ✨ Features  
- 📝 Create, edit & delete blogs  
- 🔄 Live blog updates  
- 👤 User authentication & profiles  
- 🌙 Modern UI with dark mode  
- ⚡ Fast and responsive design  

---

## 🛠 Tech Stack  
- **Frontend:** React, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Other:** REST APIs, JWT Auth
